    • 使用 HEXO + Github 建立博客

    为了建立博客，无意中看到 hexo 好评很多，就找来别人的成功经验，自己搭建基于 Github 的博客了～

    以下是自己建立博客的过程。

    原料

    • Github.com 的注册账号
    • 本地 node 环境
    • 本地 hexo 模块
    • 本地 Markdown 编辑器

    登录（Github）建立仓库（Repository），仓库命名使用 xxuyou.github.io 。
